# SOHO IoT Platform Documentation

This repository documents the architecture, configuration, and operations of the SOHO IoT platform, including services for data ingestion, device management, analytics, and UI.

## Contents

- `infra/` — Infrastructure configuration (Docker, ports, Caddy, etc.)
- `services/` — Service-specific documentation (ingest, device manager, UI)
- `schemas/` — Database schema and table documentation
- `CHANGELOG.md` — Record of platform updates and changes
