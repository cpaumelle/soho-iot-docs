# SOHO IoT Platform Documentation

This repository documents the architecture, configuration, and operations of the SOHO IoT platform.

## 🎯 Vision

A robust, modular SOHO IoT stack that:

- Ingests data from diverse IoT sources (e.g., LoRaWAN)
- Supports site/floor/room-level device management
- Offers secure, containerized services
- Scales from local Pi/Mac/Linux deployments to cloud

## Folder Structure

- `infra/` — Infrastructure config (Docker, ports, proxy)
- `services/` — Service details (ingest, device manager, UI)
- `schemas/` — Database schemas and table purpose
- `CHANGELOG.md` — Version history
